# 🏠 Apartment Rental Service

> A modern, high-performance web application for apartment listings with integrated real-time booking notifications via Telegram.

[Live Demo]
`adriatic-escape.vercel.app`

---

## 🌟 Overview

This project is a streamlined platform for property rentals. It focuses on providing a smooth user experience for potential tenants and an efficient notification system for owners. When a user submits a booking request, all details are instantly routed to a dedicated Telegram bot, ensuring no lead is missed.

## ✨ Key Features

- **Dynamic Listings:** Browse available properties with a clean, responsive interface.
- **Instant Notifications:** Custom integration with **Telegram Bot API** for real-time reservation alerts.
- **Responsive UX:** Fully optimized for mobile, tablet, and desktop viewports using a mobile-first approach.
- **Modern UI:** Minimalist design built with **Tailwind CSS** and **Lucide Icons**.
- **Type Safety:** Robust codebase developed with **TypeScript** for better maintainability and scalability.

## 🛠 Tech Stack

- **Frontend:** React 18, Next.js, TypeScript
- **Styling:** Tailwind CSS
- **Notifications:** Telegram Bot API (Integration for booking processing)
- **Environment:** Vite / Modern Web Standards

## 📁 Project Structure

- `/src/components` — Reusable UI components (Modals, Forms, Property Cards).
- `/src/constants` — Application constants including `countries.ts` for location filtering.
- `/src/hooks` — Custom React hooks for form handling and Telegram API communication.
- `/src/types` — TypeScript interfaces and type definitions for data consistency.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18.x or higher)
- npm or yarn

## Getting Started

1. Clone the repository: `git clone https://github.com/Yevhenii199/adriatic-escape`
2. Navigate to the project directory: `cd adriatic-escape`
3. Install dependencies: `npm install`
4. Run the development server: `npm run dev`
